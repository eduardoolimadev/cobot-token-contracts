COBOT Token & Presale Ecosystem 🛡️🤖
This repository contains the smart contracts for the COBOT project.

🚀 Highlights
Automated Presale: High-security presale contract featuring dynamic pricing across multiple phases.

Security First: Implementation of ReentrancyGuard and individual user cooldowns to prevent bot manipulation and sandwich attacks.

Testability: Includes a MockPriceFeed contract to simulate Chainlink Oracle behavior in local environments (Hardhat/Foundry), ensuring robust testing before deployment.

🛠️ Technical Specs
Solidity: ^0.8.20

Frameworks: OpenZeppelin for standard security patterns.
